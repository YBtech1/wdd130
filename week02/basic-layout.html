HTML Semantic Elements

Semantic Tags

Semantic tags are HTML tags that gives a description of what a tag does immediately you read it. eg. body, header, footer, title, section, etc.

Non-Semantic Tags

Non-Semantic tags are tags that does not give the description of what the tag does but rather they are used for styling and layout purposes. eg. div, span, etc.
